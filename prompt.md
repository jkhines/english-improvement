RULE 1: Use Standard American English Forms

For all general spelling variations between British and American English, I will use the American English standard forms:

This includes (but is not limited to):

-our → -or (color, honor, favor, behavior, labor, neighbor, etc.)
-re → -er (center, meter, theater, fiber, etc.)
-ise/-yse → -ize/-yze (organize, realize, analyze, recognize, etc.)
-isation → -ization (organization, civilization, realization, etc.)
-ogue → -og (catalog, dialog, analog, etc.)
ae/oe → e in scientific terms (pediatric, anemia, hemoglobin, estrogen, etc.)
Single l in words like "traveled", "labeled", "canceled"
-ense rather than -ence (defense, offense, license as verb)
-er rather than -re in agent nouns (analyzer not analyser)

Rationale: American English spelling is already more phonetically consistent and is the dominant form in international communication, technology, and science.

RULE 2: Eliminate the "ough" Pattern

The "ough" letter sequence has at least 9 different pronunciations in English, making it the single most problematic pattern for learners. This rule systematically eliminates it.

Rule 2A: through → thru

Replace through with thru in all contexts:

through → thru
throughout → thruout
Rule 2B: though/although → tho/altho

Simplify these common words:

though → tho
although → altho
Rule 2C: -ought → -awt

Regularize the -ought pattern (which represents /ɔːt/):

thought → thawt
brought → brawt
fought → fawt
bought → bawt
sought → sawt
taught → tawt
caught → cawt
wrought → rawt
nought → nawt
ought → awt

Rationale: The "aw" spelling already represents this sound in: law, saw, draw, awful, dawn, yawn, etc.

Rule 2D: tough/rough/enough → tuff/ruff/enuff

Regularize words with the /ʌf/ sound:

tough → tuff
rough → ruff
enough → enuff

Rationale: The "uff" spelling already represents this sound in: stuff, bluff, fluff, buff, cuff, huff, muff, puff, scuff.

Rule 2E: cough → coff

Regularize this word:

cough → coff
coughing → coffing
coughed → coffed

Rationale: Matches the pattern in: off, doff, scoff.

Rule 2F: dough/bough → doe/bow

Regularize these words:

dough → doe
bough → bow

Note: These create homographs with existing words (doe = female deer, bow = weapon/to bend). Context disambiguates meaning.

Rule 2G: thorough/borough → thoro/boro

Regularize these words:

thorough → thoro
thoroughly → thoroly
thoroughness → thoroness
thoroughfare → thorofare
borough → boro

Rationale: "Thoro" and "boro" have historical precedent in 19th-20th century spelling reform movements and appear in place names.
